## Usage

Run this command in the root directory to start the server:
```sh
npm start
```

## Requirements

- Node 6.14.5
- MySQL 5.7

## Development

### Installing Dependencies

From within the root directory:

```sh
npm install -g webpack
npm install
```

To seed the database:
```sh
npm run start-db
```

To download placeholder images:
```sh
npm run dl-images
```

To seed the database:
```sh
npm run seed-data
```
